API Endpoints

POST /workflow
Saves or updates a workflow.
{
  "name": "mathFlow",
  "workflow": { "mul": [{ "add": [2, 3] }, 4] }
}
GET /workflow/:name
Retrieve workflow JSON by name.
GET /workflow
List all workflow names.
GET /run/:name
Runs a workflow from the database by name.
POST /run
Runs a workflow sent directly in the POST body.
{
  "add": [2, 3]
}
GET /functions
Returns a list of available function names.
